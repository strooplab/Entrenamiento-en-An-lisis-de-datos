# Data Analyst Portfolio

Repositorio de práctica y formación como Data Analyst. El foco está en el trabajo real antes que en la presentación: limpieza de datos, transformación y exploración usando SQL, Python, Excel, Tableau Public y Power BI.

---

## Proyectos

### House Selling Dataset — SQL Data Cleaning

Limpieza completa de un dataset de venta de casas usando T-SQL. El trabajo incluyó rellenar valores nulos mediante self joins, separar columnas compuestas con `SUBSTRING` y `PARSENAME`, normalizar valores inconsistentes con `CASE WHEN`, y eliminar duplicados con `ROW_NUMBER() OVER (PARTITION BY ...)`. Inspirado en el canal de Alex The Analyst.

### Dataset Cleaning & Splitting — Python

Limpieza y división de un dataset en Jupyter Notebook usando pandas. Exploración inicial, tratamiento de nulos y duplicados, transformación de columnas y segmentación del dataset en partes para análisis independiente.
